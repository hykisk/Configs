## Extension Shortcuts
- `ctl-k, v` Open Markdown with PlantUML.
- `ctl-alt-v` Paste Image to Markdown from clipboard.
- `ctl-alt-t a` Format table in markdown.
- `shift-alt-f` Format Document.
